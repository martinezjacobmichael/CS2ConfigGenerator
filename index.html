CS2 Autoexec Generator

This tool generates an autoexec.cfg file for Counter-Strike 2.

  • Place the autoexec.cfg file inside the folder (Counter-Strike Global Offensive\game\csgo\cfg) and it will be executed automatically everytime the game is launched.
  • Some settings refer to values only as (0 / 1) or (false / true). These are interchangeable.
  • Unfilled input boxes will not be added to the autoexec.cfg file.
  • The game will not crash if you enter invalid values, but it may not work as intended.

This tool is a work in progress. Many options/commands are still unavailable.
